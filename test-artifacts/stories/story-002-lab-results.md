# Story-002: View Lab Results

## User Story

**As a** patient  
**I want to** view my lab results online  
**So that** I can track my health without calling the clinic

## Description

Patients should be able to access their lab results as soon as they're available, with clear explanations of what each test means.

## Epic

Epic-001: Patient Portal - Medical Records Access

## Acceptance Criteria

- [ ] Patient can see list of all lab results sorted by date (newest first)
- [ ] Each result shows: test name, result value, reference range, status (normal/abnormal)
- [ ] Patient can filter by date range and test type
- [ ] Abnormal results are highlighted with visual indicator
- [ ] Patient can view doctor's notes/interpretations
- [ ] New results trigger email notification
